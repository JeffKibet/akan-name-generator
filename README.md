Description of Project

The Akan Name Generator is a simple web application that calculates a user’s Akan name based on their date of birth and gender.

In Ghanaian culture, children are traditionally given names depending on the day of the week they were born. This project uses JavaScript logic to determine the correct day and assign the corresponding Akan name.

Live links
You can view the website here: https://github.com/JeffKibet/akan-name-generator

The application:
- Takes a user’s birth date
- Takes gender input (Male or Female)
- Calculates the day of the week
- Displays the correct Akan name and birth day
